## python 部分载入数据

```python

def test_page():
    max_userid = 111
    page = 1
    page_num = 10

    while True:
        start = (page - 1) * page_num + 1
        end = page * page_num
        print start, end
        # process your business
        if end > max_userid:
            break
        page += 1
```

## limit offset 的实现

```python

select * from (
    select
    user_phone, userid, last_boot_time, last_order_time, row_number() over (order by userid)  as rowid
    from smart_analysis.users
) T
where rowid >= 10 and rowid <=40

```
